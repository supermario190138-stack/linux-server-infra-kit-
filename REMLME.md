## 🇺🇸 English Overview

**linux-server-infra-kit** is a production-ready collection of Linux infrastructure tools, scripts, and automation playbooks. Designed for system administrators, DevOps engineers, and developers who need reliable, scalable, and secure server baselines.

### 🚀 Key Features
- **Automated Server Hardening:** Hardening scripts for SSH, Firewall (UFW/IPTables), and user access controls.  
- **Infrastructure as Code (IaC):** Modular configurations for deploying core services rapidly.  
- **Monitoring & Logging:** Pre-configured monitoring stacks for resource tracking and alerts.
